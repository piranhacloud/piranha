piranha:stop

Full name:

cloud.piranha.maven.plugins:piranha-maven-plugin:22.9.0-SNAPSHOT:stop

Description:

This goal will stop a Piranha Core Profile that was started with the start goal.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.

Parameters

(no parameters)
