Piranha Micro

Use an alternate HTTP server

If you want to use an alternate HTTP server you can specify it using the --http argument.

The following HTTP servers are available:

  1. grizzly - the Grizzly HTTP server
  2. http - the default HTTP server (HTTP/1.1 only)
  3. netty - the Netty HTTP server
  4. undertow - the Undertow HTTP server

So if you want to use the Netty server you would use the following command-line: 


  java -jar piranha-micro.jar --war your_webapplication.war --http netty
