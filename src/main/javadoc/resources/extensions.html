Extending Piranha using extensions

Variant extensions

The following extensions are variant extensions as they are used to deliver specific support for each of our distributions.

  1. Standard - delivers regular Servlet support (akin to Jetty and Tomcat)

Non-variant extensions

  • Annotation Scanning extension - which delivers annotation scanning support
  • Apache Commons FileUpload extensions - delivers file upload functionality using Apache Commons FileUpload
  • Async extension - delivers async support
  • Byte Stream Handler - delivers ByteStreamHandler support
  • Locale encoding extension - which delivers support for <locale-encoding> web.xml
  • ServletContainerInitializer extension - which delivers ServletContainerInitializer support
  • TEMPDIR extension - which delivers TEMPDIR web application support
  • Web.xml extension - which delivers web.xml processing support
