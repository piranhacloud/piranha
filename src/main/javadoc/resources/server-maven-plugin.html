Piranha Server

Maven Plugin

The following goals are available:
  • run
  • start
  • stop

run

This goal will deploy the Maven WAR module, start Piranha Server and wait for it. It echoes the Piranha Server console back to you for your convenience.

start

This goal will deploy the Maven WAR module and start Piranha Server in a separate process.

stop

This goal will stop a Piranha Server that was started with the start goal.
