Plugin Documentation

Goals available for this plugin:

Goal Description
piranha:help Display help information on piranha-maven-plugin.
Call mvn piranha:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
piranha:run This goal will deploy the Maven WAR module, start Piranha Core Profile and wait for it. It echoes the Piranha Core Profile console back to you for your convenience.
piranha:start This goal will deploy the Maven WAR module and start Piranha Core Profile in a separate process.
piranha:stop This goal will stop the Piranha runtime that was started with the start goal.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK 17
Memory No minimum requirement.
Disk Space No minimum requirement.
